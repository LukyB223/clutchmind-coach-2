Planned user experience

Player-authorized post-game analysis

The product is designed for review after matches are finished. It does not provide real-time tactical information during live games.

1

Player sign-in

The player authorizes access to their own match history through Riot Sign On if required.

2

Match import

The app imports completed match history, agents, maps, round results, K/D/A, ACS, and headshot percentage.

3

Pattern analysis

ClutchMind identifies repeated issues such as early deaths, poor trading, utility misuse, late rotations, and weak post-plants.

4

Personal coaching

The player receives a written coaching report with best role, best agents, weaknesses, and focused training goals.

Requested Riot API usage

APIs used only for post-game review

ClutchMind uses Riot data to analyze the authorized player's own completed matches. The app does not use Riot API data for live in-game decision making or opponent scouting.

ACCOUNT-V1 Lookup Riot ID and PUUID so the app can identify the authorized player.
VAL-CONTENT-V1 Load VALORANT agent and map metadata for readable reports.
VAL-MATCH-V1 Import completed match history and match details for post-game analysis.
RSO Used if required so players explicitly authorize access to their own personal match history.

Competitive integrity

No live gameplay advantage

What ClutchMind does

  • Analyzes completed matches after the game
  • Shows personal performance patterns
  • Recommends best role and agents from the player's own data
  • Creates post-game coaching reports and improvement goals
  • Supports optional VOD review after the match

What ClutchMind does not do

  • No real-time overlays or live tactical calls
  • No memory reading or VALORANT client interaction
  • No anti-cheat bypassing
  • No automated gameplay interaction
  • No opponent scouting during live matches

Current status: local working prototype. Production version is intended to use Riot-approved API access and player authorization for personal post-game coaching.
